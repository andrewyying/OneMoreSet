# OneMoreSet
Curate your own workout timer

## Run the app
- Install deps: `npm install`
- Start dev server: `npm start`

## Build (TestFlight)
- Configure EAS: `npm install -g eas-cli` and run `eas login`
- Build: `eas build -p ios --profile production`
- Submit: `eas submit -p ios --latest --profile production`
